## Usage:
- Build files :
```bash
$ npm i
$ npm run build
```
Then generate `fyj.min.css` and `fyj.min.js` and their sourcemap files in `dist` folder.
- Import the file in your html file :
```html
<div id="fyj"></div>

<link rel="stylesheet" href="./fyj.min.css">
<script src="./fyj.min.js"></script>
```
